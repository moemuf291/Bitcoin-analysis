Bubble Map Help
How to use the Bubble Map:
  1. Enter a Bitcoin address in the input field
  2. Click "Analyze" to start the visualization
  3. View the bubble map showing transaction flows
  4. Switch to the network graph for a different view
Understanding the Visualization:
  • Bubble Size: Represents the number of transactions
  • Color: Blue = Main address, Green = Positive net flow, Red = Negative net flow
  • Position: X-axis = Net flow (received - sent), Y-axis = Total transaction volume
  • Connections: Lines show relationships between addresses
Tips:
  • Use sample addresses to see examples
  • Hover over bubbles for detailed information
  • The analysis may take a few moments for addresses with many transactions
